# Web-Vlunerability-Sacnner
SecureScan is a Flask-based web vulnerability scanner that detects common security issues like SQL Injection, XSS, insecure headers, phishing links, and open ports. The project also integrates SQLMap and Wfuzz for advanced security testing and generates professional PDF reports with scan history storage using MongoDB.

Objective of the Project

The main objectives of this project are:

To detect common web vulnerabilities.
To automate basic penetration testing tasks.
To provide a simple interface for website security analysis.
To integrate popular security tools like SQLMap and Wfuzz.
To generate downloadable vulnerability reports.
To maintain scan history using MongoDB.
